# 🏗️ Landing Page - Строительная компания

Современный лендинг для строительной компании с калькулятором стоимости и интеграцией с Supabase.

## 🌐 Демо

[Посмотреть сайт](https://landing-y3sq.vercel.app/)

## 🚀 Технологии

- **Next.js 16** - React фреймворк
- **TypeScript** - типизация
- **Tailwind CSS** - стилизация
- **Supabase** - база данных и бэкенд
- **Vercel** - хостинг и деплой

## 📦 Функционал

- ✅ Адаптивный дизайн
- ✅ Калькулятор стоимости дома
- ✅ Форма обратной связи
- ✅ Галерея выполненных работ
- ✅ Интеграция с Supabase для хранения заявок
- ✅ Оптимизированные изображения

## 🛠️ Установка и запуск

```bash
# Клонировать репозиторий
git clone https://github.com/wnahamon/Landing.git

# Установить зависимости
npm install

# Запустить локально
npm run dev
```
